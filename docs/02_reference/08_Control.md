# Control(制御)

Controlは次のメソッドを持ちます。

## 共通.Control

<table>
<thead>
<tr><th>分類</th><th>意味</th></tr>
</thead>
<tbody>
<tr><td>Control.wait</td><td>条件成立まで待つ</td></tr>
<tr><td>Control.waitUntil</td><td>条件成立する間、待つ</td></tr>
<tr><td>Control.waitWhile</td><td>全てのスレッドを停止する</td></tr>
<tr><td>Control.stopThisScript</td><td>このスレッドを停止する</td></tr>
<tr><td>Control.stopOtherScripts</td><td>他のスレッドを停止する</td></tr>
</tbody>
</table>

## sprite.Control

<table>
<thead>
<tr><th>分類</th><th>意味</th></tr>
</thead>
<tbody>
<tr><td>Control.clone</td><td>クローンを作る</td></tr>
<tr><td>Control.removeClone</td><td>クローンを削除する</td></tr>
<tr><td>Control.removeAllClone</td><td>すべてのクローンを削除する</td></tr>
</tbody>
</table>
