# Looks(見た目)

Looksは次のメソッドを持ちます。

## sprite.Looks

- Looks.costume (コスチューム操作)
- Looks.backdrop (背景操作)
- Looks.size (サイズ操作)
- Looks.effect ( 画像効果操作 )
- Looks.layer (階層操作)
- Looks.bubble ( フキダシ操作 )
- Looks.visible ( 表示/隠す )

## stage.Looks

- Looks.backdrop (背景操作)
- Looks.effect ( 画像効果操作 )
