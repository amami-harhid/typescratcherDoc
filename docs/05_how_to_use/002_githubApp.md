# Github アプリ

### 『GitHubをインストール』

Chromeの場合、『GitHub でログインしている状態』で、<br>
『キャスト、保存、共有』→ 『GitHubをインストール』をします。<br>

<img class="block" src="/github_app_install.png" style="width:80%;height:600px;"/>

---

### 『GitHubをインストール』が見つからない場合

すでにGitHubアプリを作成済である場合、『GitHubをインストール』は表示されません。<br>
Windowsの設定⇒アプリより『GitHub』をアンインストールしてください。

---

### **Github.com** を開く

『GitHub』のショートカットが作成されていることを確認します（通常はデスクトップに作成されています）。
