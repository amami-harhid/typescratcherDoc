# Vite による実行

『Vite』により、『Typescript』のコンパイルと実行（ブラウザ表示）を行います。

『Vite』によるブラウザ表示中に スクリプトを変更して保存すると、ブラウザへ変更内容が通知され、表示中のページがリロードされます。

### (1)『npm run vite』

ターミナルより 『npm run vite_open』を実行します。

### (2) ブラウザ表示

ブラウザにページが表示されたらURLの最後にパスを入れます。

パスは、『src』の下のディレクトリ階層で『index.html』がある場所になります。

最後にスラッシュ(/)を必ず入れてください。
