# typescratcherDoc

これは、「TypeScratcherの解説サイト」です。

VitaPressを使って作成しています。

[公開サイトはこちらです](https://amami-harhid.github.io/typescratcherDoc/web/)

# npm scripts

```
npm run docs:dev
```

ビルド無しで閲覧確認ができます。

```
npm run docs:build
```

ビルドして、生成結果を `/web` へ出力します。
